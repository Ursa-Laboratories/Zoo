# Progress Notes

Use this directory for shared task progress notes that survive handoffs between agents.

Suggested format for each task note:

```md
# <task title>

Status: in_progress | blocked | complete

## Goal
- What is being changed

## Findings
- Important discoveries

## Files
- Paths touched or likely to be touched

## Verification
- Checks run

## Next Steps
- Remaining work or follow-up
```

Keep notes short, update them in place, and avoid duplicates for the same task.
